## Pré-requis

1. Télécharger et installer [uv](https://docs.astral.sh/uv/getting-started/installation/).
2. Télécharger ou cloner le dépôt git https://github.com/kaisersly/parse-s13.
  - Télécharger le dépôt :
    1. Aller sur [https://github.com/kaisersly/parse-s13].
    2. Extraire le fichier .zip dans un répertoire.
  - Cloner le dépôt : `git clone https://github.com/kaisersly/parse-s13`.
3. Dans le répertoire de l'application, lancer la commande `uv sync`.

## Lancement

1. Télécharger le répertoire S13 et le copier dans le répertoire de l'application.
2. Dans le répertoire de l'application, lancer la commande `uv run parse_s13.py`.
